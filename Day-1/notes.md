## Day 2 Numbers, Variables And Stings

# Variables
- Variables in python are created by assigning values to them
- EG : my_number = 3

# Numbers
- Numbers are variables
- Numbers are integers, floats, double
- Eg : Integer "x = 3"
- float "x = 3.5"
- Double "x = 4.54533535"

# Strings
- These are a sequence of characters surrounded by double or single quotes
- EG : x = "john doe"
- x = 'john doe'
